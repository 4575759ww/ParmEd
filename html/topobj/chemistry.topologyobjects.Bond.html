chemistry.topologyobjects.Bond¶

class chemistry.topologyobjects.Bond(atom1, atom2, type=None)[source]¶

A covalent bond connecting two atoms.

Parameters:

atom1 : Atom

The first atom involved in the bond

atom2 : Atom

The other atom involved in the bond

type : BondType

The bond type that defines the parameters for this bond

Notes

You can test whether an Atom is contained within the bond using the in operator. A BondError is raised if atom1 and atom2 are identical. This bond instance is append`ed to the `bonds list for both atom1 and atom2 and is automatically removed from those lists upon garbage collection

Examples

>>> a1, a2 = Atom(), Atom()
>>> bond = Bond(a1, a2)
>>> a1 in bond and a2 in bond
True
